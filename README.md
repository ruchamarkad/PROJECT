Lung Cancer Detection Using AI with Medication
#Goal
1.Develop an AI system to detect lung cancer from medical images.
2.Enable early detection of lung cancer.
3.Assist doctors and radiologists in diagnosis.
4.Reduce manual analysis time.
5.Improve accuracy using deep learning techniques.

#Steps Followed

1.Collected lung CT scan or X-ray image datasets.
2.Preprocessed images (resizing, normalization, noise removal).
3.Split the dataset into training and testing sets.
4.Built a deep learning model using Convolutional Neural Networks (CNN).
5.Trained the model with the training dataset.
6.Evaluated the model using performance metrics.
7.Tested the model on new medical images for prediction.
